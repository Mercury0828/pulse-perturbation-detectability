# PDET — Pulse-perturbation Detectability

A restricted-access detectability workflow for pulse-level coherent errors in quantum control.

Given a control model and a measurement budget, PDET decides, for each candidate coherent error, whether it is
visible, hidden by the readout, or hidden by the control; returns a low-cost experiment that would expose a chosen
hidden one; and reports the finite-shot cost of confirming it. The diagnosis is a classical calculation on the
control model and spends no device shots. PDET reuses the known quantum observability map and the known averaging
effect of dynamical decoupling; the contribution is the integration into a budget-aware engineering workflow, not a
new physical principle.

This repository holds the simulation code and the regenerated results for the manuscript *"Which Coherent Errors
Can You See? A Restricted-Access Detectability Workflow"* (IEEE Transactions on Quantum Engineering).

## Repository layout

- `numerics/` — the simulation code and the scripts that regenerate every figure and table.
  - `pdet_core.py` — the validated core (toggling generators, restricted response matrix, kernel, margins).
  - `models.py` — transmon DRAG and cross-resonance control models.
  - `run_all.py` — single entry point that reproduces the full result set.
  - `requirements.txt` — pinned dependencies (Python 3.12; NumPy, SciPy, QuTiP, Stim, Matplotlib).
- `results/` — regenerated data (JSON, fixed seeds) and figures (PNG).

## Reproducing the results

```bash
cd numerics
pip install -r requirements.txt
python run_all.py
```

Every figure is regenerated by a single named script with a fixed random seed (`20260628`). A correctness gate
validates the closed-form response against exact propagation (maximum relative error `1.5e-10`) before any result
is trusted. Each result maps to one script: the invisibility decomposition, the open-system blind spot, the
idle-qubit use case and its protection/detectability trade, the finite-shot scaling, the locality-scaled runtime,
the two-qubit cross-resonance verdict, and the model-mismatch sensitivity.
